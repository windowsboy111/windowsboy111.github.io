pyTableMaker

The library / module for python to allow programmers to create, edit and show tables without gui conveniently.


This library exists because Merlin needs a way to create tables conveniently, and I also want it to be challenging so I code my version of it. It creates tables dynamically, also open source.

The newest pre-release 1.3-a2 supports cell wrapping, it's essentially a way to allow words to be aligned to multiple lines properly. Users can also have a choice of making a CustomTable, and users can specify what characters to form the table. pyTableMaker does everything for you as it dynamically generates the output.

Usage

You can install pyTableMaker with pypi. python3 -m pip install pyTableMaker. Easy, right?
You can just import the library easily with import pyTableMaker.

You can check out the usage on the main Github page by clicking on the Github button on the top navigation bar. All the functions and methods are listed under README.md.

Check out CHANGELOG.md!
Check out how it performs by running example.py!
